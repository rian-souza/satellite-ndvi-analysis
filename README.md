# Satellite NDVI Analysis

Projeto de análise de vegetação utilizando **NDVI (Normalized Difference Vegetation Index)** a partir de imagens de satélite.

✅ **Status:** Concluído — Aberto para melhorias.

## Objetivo

Explorar dados de satélite para calcular e visualizar o **NDVI**, um índice amplamente utilizado para monitoramento da saúde da vegetação, demonstrando técnicas de análise de dados aplicadas ao setor agrícola.

## Tecnologias utilizadas

* Python
* Jupyter Notebook
* NumPy
* Matplotlib
* Rasterio (ou bibliotecas geoespaciais equivalentes)

## Estrutura inicial do projeto

```
satellite-ndvi-analysis/
│
├── data/            # Dados e imagens de satélite
```
